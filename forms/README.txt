Demographics, contact information, feedback, payment, information requests
* What types of tags can you use to create forms?  
<input>
<label>
<option>
<form>
<select>
<text area>
<button>
<legend>
<datalist>
* What are the benefits / disadvantages of using some tags over others? 
Prioritizing tags could depend on what is intended for the user. Using option values might be more useful for dates, whereas using a list might be more useful for multiple selections or filters. Lists and option values could also be useful to direct specific data to a new corresponding form. 
* How do you start to programmatically use the data provided by the user? 
Start by retrieving the data and utilize it for its intent.
